TrueHub is an open-source Android application designed to provide a seamless and intuitive mobile experience for interacting with your TrueNAS SCALE server. Monitor performance, manage applications, and stay on top of alerts directly from your phone.

## Features
1. App management functions:
     1. Update
     2. Start / Stop
     3. View Details
     4. Rollback
2. Container management functions
     1. Start /Stop
     2. View Details
3. VM management functions
     1. Start /Stop VMs
     2. View Details
     3. Shutdown VMs


## Getting Started

### For Users

The easiest way to get started is by downloading the latest release.

1.  Go to the **[Releases Page](https://github.com/Imnotndesh/TrueHub/releases)**.
2.  Download the latest `.apk` file.
3.  Open the file on your Android device and follow the installation instructions. You may need to grant permission to install apps from unknown sources.

### For Developers

If you wish to build the app from source:

1.  **Clone the repository:**
2.  **Open it in Android Studio**
3.  **Build if needed after syncing files with gradle**
